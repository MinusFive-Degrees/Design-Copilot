# Design Copilot

Design Copilot 是一個面向嘉立創 EDA 的統一工作台擴展，用來把原理圖檢查、PCB 檢查、網路排查、報告回看與 AI 輔助分析集中到同一個 GUI 中。

## 專案作用

這個專案不是要取代 EDA 編輯器本身，而是為設計流程增加一層快速體檢與復盤能力：

- 在原理圖階段快速發現位號、封裝、BOM 準備度與 DRC 風險
- 在 PCB 階段統計器件、焊盤、過孔、走線、覆銅與熱點網路
- 用統一報告保留多輪檢查結果，方便回看與比較
- 用 AI Agent 依據最近報告生成摘要、整改建議、複核清單與自訂問答

## 目前功能

- 統一工作台
  首頁、原理圖、PCB 功能表都只保留 `打開工作台`。所有檢查動作、網路工具、報告查看、參數設定與 AI 助手都集中在同一個 GUI 中，不再分散在多個功能表項目。
- 原理圖功能
  支援原理圖綜合檢查、快速體檢、原理圖 DRC 與選區快照。體檢會統計器件、導線、總線、文字、網路標識、位號前綴分布、BOM 準備度、缺位號與缺封裝數量，並輸出設計評分與建議動作。
- PCB 功能
  支援 PCB 綜合檢查、快速體檢、PCB DRC、選區快照與熱點網路分析。體檢會統計器件、焊盤、過孔、走線、圓弧、覆銅、填充、區域與文字數量，分析熱點網路，評估供應鏈料號完整度與過孔風險，並輸出設計評分。
- 網路排查工具
  在 PCB 上下文中，可以自動高亮最密網路，也可以從熱點網路列表中點選高亮，或依網路名稱直接聚焦，適合排查電源網、地網與關鍵高速網路。
- 報告系統
  每次綜合檢查、快速體檢、DRC 與選區快照都會生成統一格式的報告。工作台內建報告舞台顯示最近結果，並保留最近 8 次歷史記錄，方便多輪修改後回看與比較。
- 參數維護
  可以在 GUI 中直接維護 `Top 熱網數量`、`熱網閾值`、`選區預覽數` 與 `過孔風險閾值`。這些參數會直接影響報告中的熱點網路排行、快照預覽長度與 PCB 風險提示。
- AI 助手
  支援填寫自訂接口地址、模型名、API Key、額外請求頭 JSON、系統提示詞與 Temperature。內建 `總結最近報告`、`生成整改建議`、`生成複核清單` 與 `自訂問答` 四種動作，模型呼叫時會自動附帶最近報告、當前文檔上下文、熱點網路與目前參數。

## 數據來源

- 設計統計來自 `eda.sch_*`、`eda.pcb_*` 與 `eda.dmt_*` 系列 API
- 最近報告、歷史記錄、參數與 AI 設定保存在 `eda.sys_Storage`
- AI 請求透過 `eda.sys_ClientUrl.request` 發往自訂模型接口

> 使用 AI 助手前，需要在嘉立創 EDA 中為擴展開啟「外部互動」權限，並確認目標接口允許 CORS。當前請求格式依照 OpenAI 相容的 `chat/completions` 組織。

## 開發與打包

```bash
npm install
npm run build
```
