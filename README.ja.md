# Design Copilot

Design Copilot は、嘉立創 EDA 向けの統合ワークベンチ拡張です。回路図チェック、PCB チェック、ネット解析、レポート確認、AI 支援分析を 1 つの GUI に集約します。

## このプロジェクトの役割

このプロジェクトは EDA 本体を置き換えるものではありません。設計フローに対して、素早い点検と振り返りのレイヤーを追加することが目的です。

- 回路図段階で、Designator、フットプリント、BOM 準備度、DRC リスクを早く見つける
- PCB 段階で、部品、パッド、ビア、配線、銅箔、密集ネットを集計する
- 複数回のチェック結果をレポートとして保持し、比較しやすくする
- AI Agent を使って、最新レポートの要約、修正提案、レビュー項目、自由質問を生成する

## 現在の機能

- 統合ワークベンチ
  ホーム、回路図、PCB の各メニューでは `Open Workbench` だけを表示します。検査アクション、ネット解析、レポート確認、設定、AI 機能をすべて同じ GUI に集約し、複数メニューに分散させません。
- 回路図機能
  総合チェック、クイック監査、回路図 DRC、選択範囲スナップショットを提供します。監査では部品、配線、バス、テキスト、ネット識別子、Designator 接頭辞分布、BOM 準備度、未設定 Designator、未設定フットプリントを集計し、スコアと推奨アクションを出力します。
- PCB 機能
  総合チェック、クイック監査、PCB DRC、選択範囲スナップショット、密集ネット解析を提供します。監査では部品、パッド、ビア、配線、円弧、ポア、フィル、リージョン、テキストを集計し、ホットネット、部品サプライチェーン情報の欠落、ビアリスクを評価してスコア化します。
- ネット解析ツール
  PCB コンテキストでは、最密ネットの自動ハイライト、現在のホットネット一覧からの選択ハイライト、ネット名による直接フォーカスが可能です。電源網、GND、高速配線の確認に向いています。
- レポート機能
  総合チェック、クイック監査、DRC、選択範囲スナップショットのたびに統一形式のレポートを生成します。ワークベンチ内のレポート表示エリアで最新結果を確認でき、直近 8 件の履歴を保持して比較できます。
- パラメータ管理
  GUI から `Top Net Count`、`Dense Net Threshold`、`Selection Preview Limit`、`Via Risk Threshold` を直接編集できます。これらの値はホットネット順位、スナップショット表示量、PCB リスク提示に反映されます。
- AI アシスタント
  カスタムエンドポイント、モデル名、API Key、追加ヘッダー JSON、システムプロンプト、Temperature を設定できます。`最新レポート要約`、`修正提案`、`レビュー項目生成`、`自由質問` を用意しており、各リクエストには最新レポート、現在の文書コンテキスト、ホットネット、現在の閾値が自動で含まれます。

## データソース

- 設計統計は `eda.sch_*`、`eda.pcb_*`、`eda.dmt_*` 系 API から取得
- 最新レポート、履歴、設定、AI 構成は `eda.sys_Storage` に保存
- AI リクエストは `eda.sys_ClientUrl.request` を通して送信

> AI アシスタントを使う前に、嘉立創 EDA 側でこの拡張の「外部連携」権限を有効化し、対象エンドポイントで CORS を許可してください。現在のリクエスト形式は OpenAI 互換の `chat/completions` です。

